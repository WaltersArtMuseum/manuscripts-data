
Supplemental Data: Digitized Manuscripts from the Walters Art Museum
====================================================================

The data files in this repository provide additional data, to accompany [a repository of manuscripts data at OPENN](https://openn.library.upenn.edu/html/0054.html) (and originally at [The Digital Walters](https://thedigitalwalters.org/)). The data is also used to present a web site, [The Walters Ex Libris](https://manuscripts.thewalters.org/).

The data describes Rare Books and Manuscripts at The Walters Art Museum, a collection that includes almost 900 beautifully illuminated manuscripts, about 1,300 incunabula, and around 2,000 rare books. The collection contains codices from around the world and spans over 1,000 years, from the 9th to the 20th century.
